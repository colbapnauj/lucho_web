# Landing Page - Arquitecto

Landing page moderna para arquitecto con menú de navegación interactivo usando JavaScript vanilla.

## Características

- Menú de navegación con 5 secciones principales
- Submenús desplegables (2-3 opciones cada uno)
- Diseño responsive
- Logo a la izquierda
- Botones de acción a la derecha
- Sin frameworks, solo HTML, CSS y JavaScript vanilla

## Estructura

- `index.html` - Estructura HTML
- `styles.css` - Estilos CSS
- `script.js` - JavaScript vanilla

## Uso

Simplemente abre `index.html` en tu navegador o usa un servidor local:

```bash
# Con Python
python -m http.server 8000

# Con Node.js (http-server)
npx http-server

# Con PHP
php -S localhost:8000
```

Luego abre `http://localhost:8000` en tu navegador.
